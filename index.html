      }
    }
  }
}

29. Advanced Concepts

Master custom inline arbitrary layouts (w-[380px]), reusing styles via directives (@apply), and baseline layers.

Arbitrary Measurements inline

Configure precise lengths directly inside utility tags using brackets: w-[380px], h-[calc(100vh-80px)], or bg-[#ff00aa].

Class bundling using @apply

Bundle classes directly inside custom CSS sheets to keep layouts cleaner:

.btn-primary {
  @apply px-6 py-3 bg-blue-600 text-white rounded-full font-semibold;
}
